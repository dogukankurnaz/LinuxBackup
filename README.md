# LinuxBackup

It backs up your files and saves it in /tmp.

## Installation

Install dependencies and dev Dependencies and start the script.

```
git clone https://github.com/dogukankurnaz/LinuxBackup.git
chmod +x backup.sh
./backup.sh

```
